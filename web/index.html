描述發生了什麼事

用平常說話的方式描述事由，系統會在台灣刑事判決中找出事實相似的案例，整理過去法院怎麼判、賠償落在什麼範圍、理由是什麼。

資料範圍：民國 105–114 年地方法院刑事判決，以過失傷害、公共危險、肇事逃逸三類案件為主。

直接以檔案（file://）開啟無法載入資料。請在 repo 根目錄執行 uv run python -m http.server 8080，再開啟 http://localhost:8080/web/。

    關於本系統

    • 這是什麼、不是什麼

      這是一個類案檢索工具：幫助你了解與自身情況相似的案件，過去法院如何判決。它不是法律建議，不對任何個案做預測；重要決定請諮詢律師或法律扶助基金會。

    • 資料範圍與已知偏差

      資料來自司法院開放資料平臺。請注意選擇性偏差：純財物損壞（如僅後照鏡受損）在刑事上多以不起訴收場，幾乎不會出現在判決書中，因此檢索結果會偏向起訴後、後果較重的案件。系統在每筆抽取欄位附上信心標記，無法對應原文的結論會以紅底標出。

    • 檢索如何運作

      四個步驟：釐清要件、推斷可能法條、混合檢索（語意向量 BGE-M3 加上關鍵詞 BM25，以 RRF 融合）、再以 bge-reranker 重排取前五件。每次檢索都可展開「檢索過程」查看實際步驟，細節見 repo 內 docs/design_v1.md。

    資料來源：司法院開放資料平臺（opendata.judicial.gov.tw）。本網站為「生成式 AI 的人文導論」課程期末專題，僅供研究與教學展示。

    本系統為類案檢索工具，非法律建議；過去案例不代表您的案件結果。
